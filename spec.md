# Riley - AI Coding Assistant Chatbot

## Overview
Riley is an AI chatbot application designed for coding assistance, homework help, and general Q&A. The application features a dual-interface design combining chat functionality with an integrated code editor.

## Core Features

### Dual Interface
- **Chat Interface**: Primary conversational area with futuristic-minimal design featuring soft gradients, fluid animations, and subtle glowing accents
- **Code Editor Panel**: Integrated Monaco-style code editor for writing, running, and testing code snippets within the chat interface

### Chat Functionality
- Real-time conversational interface simulating intelligent AI responses
- Support for multiple topic areas: coding assistance, academic help, and general knowledge questions
- Message animations and smooth transitions
- Chat history display showing previous conversations
- Dynamic UI reactions during Riley's "thinking" and response phases

### Visual Design
- Futuristic-minimal aesthetic with soft gradients and fluid motions
- Subtle glowing accents throughout the interface
- Theme toggle between light and futuristic dark modes
- Responsive layout optimized for both desktop and mobile devices
- Immersive user experience with engaging animations

### Technical Requirements
- All chat conversations and history stored in backend
- Real-time message processing and response generation
- Code execution capabilities within the integrated editor
- Responsive design supporting various screen sizes
- English language interface

## Backend Data Storage
- Chat conversations and message history
- User preferences including theme settings
- Code snippets and execution results

## Backend Operations
- Process and respond to user messages across different topic categories
- Store and retrieve chat history
- Handle code execution requests from the integrated editor
- Manage user theme preferences
